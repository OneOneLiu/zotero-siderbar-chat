RAG (Retrieval-Augmented Generation)
Papers are split into chunks and indexed locally. In follow-up questions, only relevant chunks are sent to the AI instead of the full text, saving tokens significantly.
Number of most-relevant text chunks to retrieve per follow-up question. More chunks = more context but higher token cost.

Synthesis Prompt
After all papers are extracted, this prompt drives the cross-paper synthesis. Use {question} for the user's question and {count} for the number of papers.
