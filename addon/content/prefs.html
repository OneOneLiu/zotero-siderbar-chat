Pipeline Overview
The analysis runs in 4 phases:
① RAG Index (local) → ② Question Understanding (AI) → ③ Per-paper Extraction (AI) → ④ Synthesis (AI)
Then ⑤ Follow-up Questions use RAG + chat history for continued analysis.
Customize the prompts for phases ②③④⑤ below.
② Question Understanding Prompt
Before extracting from papers, the AI first analyzes the research question to identify core concepts, clarify ambiguities, and generate sub-questions. This understanding document guides all subsequent steps. Use {question} as a placeholder for the user's research question.
③ Per-paper Extraction Prompt
This prompt is sent to the AI for each paper to extract structured information, guided by the question understanding from phase ②. Use {question} for the user's question and {understanding} for the question analysis output.
④ Synthesis Prompt
After all papers are extracted, this prompt drives the cross-paper synthesis. Use {question} for the user's question, {understanding} for the question analysis, {extractions} for all per-paper extraction results, and {count} for the number of papers.
⑤ Follow-up Question Prompt
After the initial analysis, this prompt wraps every follow-up question to maintain rigor and consistency. The AI receives this prompt along with the question understanding, previous analysis, RAG passages, and chat history. Use {question} for the user's follow-up question.
